# PowerShell

### .NetFramework3.5有効化.ps1
Windows PowerShell を使用した .NET Framework 3.5 の有効化

### ファイル削除.ps1
指定したフォルダ（第1引数）内にある指定したファイルを削除する。<br>
サブフォルダまで対象ファイルがあるか確認します。
### ファイル名変更.ps1
指定したフォルダ（第1引数）内にあるファイルに対し、ファイル名を一括置換します。
